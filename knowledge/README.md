# Cross-Project Knowledge Base

Cross-project knowledge for your Claude Code collaboration. This directory contains analytical patterns, conventions, and lessons learned that apply across all projects.

Install to `~/claude-knowledge/` (or wherever your `CLAUDE.md` points to).

## Files

| File | Contents |
|------|----------|
| `conventions.md` | Process rules and workflow conventions |
| `stata_patterns.md` | Stata-specific technical patterns |
| `methods_library.md` | Analytical methods: setup, diagnostics, interpretation |
| `lessons.md` | Trial-and-error lessons and debugging insights |

## Usage

Referenced from `~/.claude/CLAUDE.md`. Each project's `CLAUDE.md` should point here for cross-project knowledge. The agents (methods-referee, coder-critic) read these files before every review.
